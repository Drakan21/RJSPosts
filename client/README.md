## Overview

### Functionality Provided

-   Authentication (Login/Validation) for user activity
-   Permit users to:
    -   Post blog entries
    -   Edit blog entries (if permitted)
    -   Remove blog entries (if permitted)
-   Browse all other posts
-   View user profile, and associated posts.
